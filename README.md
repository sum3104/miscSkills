# miscSkills

業務効率化のためのエージェント用スキル（Agent Skills）を試すためのリポジトリです。
スキル一式ごとに `YYYYMMDD_名前/` のサブフォルダに置いています。詳細は各フォルダの README を参照してください。

A repository for experimenting with agent skills (Agent Skills format) aimed at improving work efficiency.
Each skill set lives in its own `YYYYMMDD_name/` subfolder; see the README in each folder for details.
